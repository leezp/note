CS的stage马上线时，默认会向一个符合checksum8规则的路径发起请求，随后服务器会响应各种Payload数据。checksum8规则路径大概就长这样：

	/x9cI/
	/fYKR/
	/Mrm0/
	/wQPD/
	/yDHX/
	/BCre/
	/WHVh/

利用checksum8规则的特点给CS服务器疯狂假上线干扰攻击队，或者直接给各大杀软和沙箱批量交样本。
